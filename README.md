# 🤖 HexSoftwares - Emotion Detection AI System

## 📌 Project Overview
This project is developed under **HexSoftwares**.  
It is an AI-powered Emotion Detection system that uses Machine Learning and Natural Language Processing (NLP) to detect emotions from text input.

The model can classify emotions such as **joy, sadness, anger, love, fear, and surprise** with confidence scores and displays results in an interactive Streamlit web application.

---

## 🚀 Features
- 🧠 Emotion detection from text input
- 🤖 Machine Learning model (Logistic Regression)
- 📊 Confidence score for predictions
- 😊 Emoji-based emotion output
- 📜 Prediction history tracking
- ⬇️ Download results as CSV file
- 🎨 Interactive Streamlit web interface
- ⚡ Real-time prediction

---

## 🧠 Tech Stack

## 📊 Dataset Information
This project uses a publicly available emotion dataset for training the model.

### Dataset Link:
https://www.kaggle.com/datasets/praveengovi/emotions-dataset-for-nlp

### Dataset Details:
- Total Samples: ~16,000+ text entries  
- Type: Text classification dataset  
- Labels: joy, sadness, anger, love, fear, surprise
