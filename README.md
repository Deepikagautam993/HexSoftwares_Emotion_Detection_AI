# 🤖 HexSoftwares | Emotion Detection AI System

## 📌 Overview
The **Emotion Detection AI System** is a Machine Learning and Natural Language Processing (NLP) based project developed under **HexSoftwares**.

This system analyzes text input and predicts human emotions such as **joy, sadness, anger, love, fear, and surprise** with probability confidence scores.  
It is deployed using an interactive **Streamlit web application** for real-time prediction.

---

## 🚀 Key Features
- 🧠 AI-based Emotion Detection from text
- 🤖 Machine Learning model (Logistic Regression)
- 📊 Confidence score for predictions
- 😊 Emoji-based emotion visualization
- 📜 Prediction history tracking
- ⬇️ Downloadable results (CSV format)
- 🎨 Interactive Streamlit UI
- ⚡ Fast real-time inference
